# Kajitter

[デモページ](kajitter.vercel.app/)

## 概要

Kajitter は家事を世界にオープンにし、みんなで家事をするモチベーションを高め合う SNS です。

## Quick Start

必要ライブラリをインストールします。

```bash
npm install
```

その後、以下のコマンドを実行することで、開発サーバーが立ち上がります。

```bash
npm run dev
```

開発サーバは 以下の URL からアクセスできます。

- http://localhost:3000
